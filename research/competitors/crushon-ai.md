# CrushOn.AI

## Snapshot

- Category: AI character chat / roleplay platform
- Company: CrushOn.AI
- Website: https://chat.crushon.ai/
- Status: Active consumer product
- Last researched: 2026-06-21

## Product Thesis

CrushOn.AI's thesis is unfiltered character roleplay with a large character library, custom character creation, memory, and credit-based access to higher-quality models.

## Core Interaction Loop

Users browse or create characters, chat or roleplay, spend credits on pro models, and upgrade for memory, limits, and priority features.

## Job-To-Be-Done

Provide flexible roleplay with fewer restrictions than mainstream character platforms.

## Memory Model

Official FAQ copy says Premium includes long-term memory and that characters can remember past conversations. The CrushOn wiki says extensive model memory helps characters adapt.

## Voice, Avatar, and Embodiment

CrushOn is mostly text/character-card-led. Some third-party reviews mention voice or image features at higher tiers, but this should be verified in-product.

## Retention Loop

Retention comes from character library, unfiltered positioning, memory continuity, daily credits, and paid tiers with higher limits.

## Pricing and Business Model

Freemium plus credit/subscription tiers. Official FAQ lists Standard, Premium, Luxe, Elite, and Imperial tiers with credits and memory differences.

## What Users Love

Users like fewer filters, broad character choice, roleplay continuity, and low entry price versus some competitors.

## What Users Complain About

Complaints include inconsistent memory, credit complexity, pricing escalation, and variable model quality.

## Strategic Lessons

CrushOn is a useful benchmark for pricing and memory gating. Companions should avoid making core trust depend on a confusing credit economy.

## Open Questions

- How reliable is long-term memory across tiers?
- Does credit pricing harm trust?
- Which parts of roleplay customization translate to non-roleplay friendship?
