# Companions

Functional Requirements:
- User should be able to have audio conversations with the companion.
- User should be able to have text chats with the companion.

- Companion should be able to interact with the internet to do research, get new data.
- Companion should have long term memory, and help the user achive their life goals.
- Companion should be able to surprise and delight the user with stories, ideas or new information.
- Companion should house a 3D body and be able to control it and use it to express themselves.
- Companion should be able to control their fake 3D environment.

Nice to haves:
- Custom 3D companions
