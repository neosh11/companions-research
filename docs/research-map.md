# Research Map

This is the first reading list for the lab. Each paper should eventually get an annotated note in `papers/`.

## AI Companions and Friendship

- [Conversational Agents: Goals, Technologies, Vision and Challenges](https://pmc.ncbi.nlm.nih.gov/articles/PMC8704682/)
  - Broad survey of conversational-agent systems.
  - Read for goals, architecture patterns, and open technical challenges.

## Trust and Retention

- Which memory failures make users stop relying on the companion?
- Which correction flows restore trust?
- Which proactive behaviors create repeated use without feeling intrusive?
- Which job-to-be-done makes the companion worth returning to?

## Storytelling and Personality

- [Storytelling Literature Review](storytelling-literature-review.md)
  - Ranked review of 30 academic sources on good storytellers, storytelling environments, narrative transportation, oral storytelling, self-disclosure, psychological safety, story circles, and human-AI storytelling.
  - Read for companion behavior, memory design, voice interaction, personality, and user storytelling loops.
- [Storytelling Synthesis By Ideas](storytelling-synthesis-by-ideas.md)
  - Cross-paper synthesis grouped by the core ideas that matter for product design: transportation, tellability, performance, identity, emotion, disclosure, safety, group setting, and human-AI agency.

## Long-Term Memory

- [Practical Considerations for AI Assistants with Long-Term Memory](https://arxiv.org/html/2409.11192v1)
  - Good overview of personal AI apps with long-term memory.
  - Read for user control, privacy, memory failures, and deployment implications.

- [Bridging Intuitive Associations and Deliberate Recall](https://aclanthology.org/2025.findings-acl.901.pdf)
  - Personal-assistant memory graph work.
  - Read for how to represent utterances, events, entities, and time.

- [LLM Agent Memory: A Survey from a Unified Representation](https://openreview.net/forum?id=KPs1EgGKcT)
  - Survey of memory mechanisms for agentic applications.
  - Read for taxonomy and evaluation ideas.

- [A Survey on Long-Term Memory Security in LLM Agents](https://arxiv.org/abs/2604.16548)
  - Recent memory-security survey.
  - Read for lifecycle risks: write, store, retrieve, execute, share, forget, and rollback.

## Embodiment and Presence

- [Systematic Review of Social Presence](https://pmc.ncbi.nlm.nih.gov/articles/PMC7805699/)
  - Useful baseline on what "presence" means and how it is measured.

- [Comparing Behavioral and Self-Report Measures of Embodied Agents](https://vhil.stanford.edu/sites/g/files/sbiybj29011/files/media/file/bailenson-comparing.pdf)
  - Useful for thinking about avatar evaluation beyond self-report.

- [Embodied Conversational Agents in Extended Reality: A Systematic Review](https://ieeexplore.ieee.org/iel8/6287639/10820123/10985757.pdf)
  - Useful survey for ECA design, presence, trust, and interaction in XR-like contexts.

## Competitive and Product Research

Track:

- Replika.
- Character.AI.
- Pi / Inflection.
- OpenAI voice experiences.
- Sesame-style voice companions.
- Hume-style voice systems.
- HeyGen and avatar pipelines.
- Game NPC systems and embodied agent research.

For each system, capture:

- Core interaction loop.
- Memory model visible to the user.
- Voice and latency quality.
- Avatar or embodiment strategy.
- Safety posture.
- Monetization and cost clues.
- What users love.
- What users complain about.
